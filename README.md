# C# WinForms x SQL Server： 沙威瑪店倉管與銷售管理系統 🌯

這是一個結合 **C# Windows Forms** 前端介面與 **SQL Server** 後端資料庫的整合專案。模擬開發一套商用的餐飲管理系統，實作完整的資料持久化與關聯式資料庫操作。

## 🌟 技術與實作功能

* **完整的 CRUD 邏輯：** 透過 ADO.NET 實作對資料庫的「新增、讀取、更新、刪除」操作，包含：
  * **顧客管理：** 新增與紀錄顧客資訊。
  * **品項動態維護：** 實作商品價格更新與下架處理。
  * **交易紀錄串接：** 紀錄每筆訂單對應的顧客與品項。
* **資料庫部署自動化：** 附帶 `.sql` 部署指令碼，展示了對環境遷移與資料庫結構的掌握能力。
* **防呆機制：** 實作基本的輸入驗證，確保資料送往 SQL Server 前的正確性。

## 📂 檔案說明
* `Practice8/`： C# WinForms 專案原始碼。
* `SQLQuery1.sql`： 用於初始化資料庫環境的 SQL 部署腳本，包含 `Customers`, `Items`, `TransactionHistory` 三張關聯表之建立。

## ⚠️ 執行環境說明
本專案原開發環境串接於遠端 SQL Server 伺服器。
**目前因課程伺服器端已關閉，直接執行會產生連線逾時錯誤。**

若需於本地端恢復運行：
1. 先在本地 SQL Server 執行 `SQLQuery1.sql` 以建立資料庫結構。
2. 修改 `Form1.cs` 中的 `ConnectionString` 連線字串，指向您的本地實例。
